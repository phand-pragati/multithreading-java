# Java Multithreading

<ul>
  <li>Threads
    <ul>
      <li>A Java Thread is an independent path of execution within a program.</li>
      <li>Many Threads can run concurrently within a program.</li>
      <li>Every Thread in Java is created and controlled by the <code>java.lang.Thread</code> class.</li>
    </ul>
  </li>
  <li>Ways
    <ul>
      <li>Extend Thread class</li>
      <li>Implement Runnable interface</li>
      <li>Anonymous implementation</li>
      <li>Executor Services</li>
    </ul>
  </li>
  <li>Synchronization
    <ul>
      <li>Volatile keyword (to avoid issue caused by caching)</li>
      <li>Synchronized keyword
        <ul>
          <li>Method</li>
          <li>Code Blocks</li>
        </ul>
      </li>
    </ul>
  </li>
  <li>Thread Pool</li>
  <li>Countdown Latch</li>
  <li>Blocking Queue</li>
  <li>Wait And Notify</li>
  <li>Re-Entrant Lock</li>
  <li>Semaphore</li>
  <li>Callable And Future</li>
  <li>Interrupting Thread</li>
</ul>
